# 需求

主要分前端订餐系统和后台管理系统。

### 后台管理系统
- 店铺管理
 - 添加店铺
- 菜单管理
 - 每个店铺对应一份菜单
 - 每份菜单包含一个或多个项目集合
 - 每个集合包含多个项目
 - 每个集合中用户可以同时选择1个或多个项目
 - 每个集合可以设置必选或可选
- 用户管理
 - 审核用户
- 订餐管理
 - 发起订餐：管理员选择店铺后即可发起订餐，并生成对应链接和二维码
 - 结束订餐
 - 查看订餐详情
 - 打印订餐详情

### 前端订餐系统
- 用户使用微信授权登陆，首次登陆填写真实姓名
- 选择菜单中的各集合中的项目提交即可
- 结束订餐前，用户可以修改选择
